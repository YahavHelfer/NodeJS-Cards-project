API Documentation

POST /users/Register

POST /users/Login

GET /users (Get all users - Admin only)

GET /users/:id (Get specific user - User only)

PUT /users/:id (Update user - Admin/User)

PATCH /users/:id/status (Change isBusiness - User only)

DELETE /users/:id (Delete user - Admin/User)

GET /cards/ (Get all cards - User only)

GET /my-cards/:id (Get my cards - User only)

GET /cards/:id (Get card - User only)

POST /cards/(Create New Card - Business User)

PUT /cards/:id (Edit Card - Business User)

PATCH /cards/:id/ (Like Card - Users Only)

DELETE /cards/:id (Delete Card - Admin/The user who created the card)
